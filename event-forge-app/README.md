# EventForge App

EventForge is a web application for discovering, organizing, and purchasing tickets for events.  
This project was built as part of a course assignment and demonstrates modern React development with reusable components, context-based state management, and responsive UI design.

---

## Features
- User authentication (login/register)
- Organizer dashboard with profile management
- Event listing and ticket purchase flow
- Shopping cart for tickets
- Context-based global state (User, Cart, Event)
- Responsive design with polished UI

## Dependencies
